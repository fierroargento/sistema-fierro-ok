Ayuda APB

Carga

Alta de pedido

1. Entrá a Nuevo Pedido.

2. Completá los datos de Cliente.

3. Completá los datos de Venta.

4. Completá los datos de Envío.

5. Cargá los Productos.

6. Guardá el pedido.

Seguimiento después de despachado

1. Revisá el tablero y seguí la Acción sugerida.

2. Si el pedido requiere confirmación de entrega al cliente, usá el botón correspondiente.

3. Si el pedido de Mercado Libre / Acordás la Entrega ya fue recibido por el cliente, usá Ya avisé Mercado Libre.

4. El pedido recién se considera cerrado cuando termina su flujo final.

Despacho

1. Revisá los pedidos del tablero de despacho.

2. Imprimí la etiqueta si corresponde.

3. Prepará y embalá el pedido.

4. Despachá el pedido.

5. No cargues seguimiento salvo que el flujo real del canal lo pida.

Mercado Libre - Acordás la Entrega

1. Cuando el pedido ya fue entregado al cliente, el sistema puede dejarlo en estado Entregado.

2. En ese punto todavía puede faltar la acción final: avisar a Mercado Libre.

3. Usá el botón Ya avisé Mercado Libre para cerrar correctamente ese tramo del flujo.

Productos

El administrador puede actualizar el padrón desde el menú Productos subiendo un Excel con:

• primera columna: SKU

• segunda columna: Descripción

La carga reemplaza el padrón actual.
